    Catálogo de productos¶

    Plan Oro

    El módulo de ventas (Catálogo, Impulsores, Ventas y Recuperación) está disponible en el plan Oro.

    Tu asistente vende lo que está en el catálogo: nombres, descripciones, precios, fotos y stock. Mientras más completo, mejor vende.

    Crear un producto¶

    1. En Catálogo pulsa Nuevo producto.
    2. Escribe el Nombre y una Descripción vendedora: es lo que el bot le contará a tus clientes.
    3. Pon el Precio y, si controlas inventario, el Stock (déjalo vacío para “sin límite”).
    4. Agrega Etiquetas para organizarte y Fotos del producto: el bot puede enviarlas en el chat.
    5. Opcional: en Recuperación de carrito puedes escribir hasta 3 mensajes propios de este producto para quien deje el pago pendiente.
    6. Pulsa Guardar.

    Importar muchos de una vez¶

    Si ya tienes tu catálogo en Excel, usa Importar CSV. El archivo debe tener las columnas name, description, price, imageUrl y tags (varias etiquetas separadas por |). Klientia crea todos los productos y te dice cuántos se crearon y si alguna fila tuvo problemas.

    Eliminar productos con ventas

    Si un producto ya tiene ventas asociadas, al “eliminarlo” se desactiva en vez de borrarse, para no perder el historial.
